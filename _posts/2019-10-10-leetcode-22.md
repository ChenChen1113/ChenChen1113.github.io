**题目描述：**  
给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。  
例如，给出 n = 3，生成结果为：  
> [
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]

**方法一、回溯**  
```python
def generateParenthesis(n):
        """
        :type n: int
        :rtype: List[str]
        """
        res = []
        
        def backtrack(s, left, right):
            if len(s) ==  2 * n:
                res.append(s)
                return
            
            if left < n:
                backtrack(s + '(', left + 1, right)
            if right < left:
                backtrack(s + ')', left, right + 1)
                
        backtrack('', 0, 0)
        return res
```
